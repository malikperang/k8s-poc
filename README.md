# Kubernetes Proof of Concept with Vagrant.

## VM Specifications

- Ubuntu 18.04
- 2 Core CPU's
- 4GB RAM on Master node
- 2GB Ram on Worker's node

## Tools Required

- Vagrant
- Docker
- Kubeadm
- Calico

This POC consists of three nodes setup.

# Usage

## Initialize server

1. Run up_server.sh

## Halt server

1. Run halt_server.sh

## Destroy server

1. Run destroy.sh
